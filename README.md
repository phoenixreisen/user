# Phoenix Reisen User

User Handler, der den typischen Umgang mit einem eingeloggten bzw. sich einloggenden Phoenix-User erleichtert.

## Installation

```bash
npm install --save @phoenixreisen/user
```

## Test

```bash
npm run test
```
